# 🎨 格子匠 (GridCraft) Pro

**格子匠** 是一款专为手工爱好者（拼豆、十字绣、像素画）设计的轻量化网页图像转换工具。它能将任何高分辨率照片转化为清晰的像素网格图，并自动匹配专业色板。

---

## 🚀 核心功能 (Core Features)

* **⚡ 智能像素化**：采用 **最邻近插值 (Nearest Neighbor)** 算法，确保缩放后像素边界清晰，无模糊感。
* **🌈 颜色聚类 (K-means)**：内置 K-means 算法，能自动从复杂照片中提取核心色彩，减少手工所需的颜色种类。
* **🧵 专业色板匹配**：支持 **Perler 拼豆** 及 **DMC 绣线** 色板模式，直接将图片颜色映射为真实的材料色号。
* **🔍 智能交互对色**：点击下方色表即可在预览图中**高亮显示**所有同色格子，彻底解决“找花眼”的痛点。
* **📄 PDF 一键图纸**：基于 `jsPDF` 库，生成包含作品预览、颜色索引及**材料用量清单（购物清单）**的可打印 PDF。
* **💾 灵活保存**：支持现代浏览器的“另存为（指定路径）”及传统的“快速下载”双模式。

---

## 🛠️ 技术栈 (Tech Stack)

* **HTML5 / CSS3**：响应式布局与自适应预览窗口。
* **Vanilla JavaScript**：核心图像处理算法（K-means, 色彩距离计算）。
* **HTML5 Canvas**：图像采样、像素级渲染及网格生成。
* **jsPDF**：前端生成多页 PDF 报告。

---

## 📖 如何使用 (How to Use)

1.  **上传图片**：点击“上传图片”选择你想要转换的照片。
2.  **设定尺寸**：在“横向格子数”输入你手工底板的宽度（高度将自动等比例缩放）。
3.  **选择模式**：
    * **自动聚类**：适合写实照片，可自定义颜色数量 $K$。
    * **固定色板**：适合专业手工，强制匹配材料标准色。
4.  **生成与调整**：点击“生成预览”，点击下方色卡核对位置。
5.  **导出结果**：点击“生成 PDF”获取打印稿，或直接保存为 PNG 图片。
